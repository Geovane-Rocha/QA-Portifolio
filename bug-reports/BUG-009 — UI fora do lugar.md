# Bug Report — BUG-009

## Título
Usuário visual_user exibe carrinho e botão Checkout fora do lugar

## Ambiente
- **SO:** Windows
- **Navegador:** Opera GX / Google Chrome
- **Dispositivo:** Computador Desktop

## Pré-condição
Usuário deve estar na página https://www.saucedemo.com/inventory.html
logado com a conta `visual_user`

## Passos para Reproduzir
1. Acessar https://www.saucedemo.com
2. Fazer login com usuário `visual_user` e senha `secret_sauce`
3. Acessar a página de checkout
4. Observar a posição do carrinho e do botão Checkout

## Resultado Esperado
O carrinho deve estar no canto superior direito e o botão
Checkout no canto inferior direito.

## Resultado Obtido
Ao entrar na página de produtos e de checkout, o carrinho
e o botão Checkout estão fora de seus lugares corretos.

## Severidade
🔴 Alta

## Prioridade
🔴 Alta

## Impacto
Impacta negativamente o usuário pois passa a impressão de
uma página mal feita, podendo fazer o usuário desistir da compra
por falta de credibilidade.
