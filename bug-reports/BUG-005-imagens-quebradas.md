# Bug Report — BUG-005

## Título
Duas imagens quebradas e uma com silhueta indicando falha

## Ambiente
- **SO:** Windows
- **Navegador:** Opera GX / Google Chrome
- **Dispositivo:** Computador Desktop

## Pré-condição
Usuário deve estar na página inicial do the-internet.herokuapp.com

## Passos para Reproduzir
1. Acessar https://the-internet.herokuapp.com
2. Clicar em **Broken Images** no menu
3. Observar as imagens exibidas na página

## Resultado Esperado
Todas as imagens devem ser exibidas corretamente.

## Resultado Obtido
2 imagens aparecem quebradas e 1 exibe apenas uma silhueta,
indicando falha no carregamento.

## Severidade
🟠 Média

## Prioridade
🟠 Média

## Impacto
Imagens quebradas prejudicam a experiência visual do usuário
e passam impressão de site mal mantido.
