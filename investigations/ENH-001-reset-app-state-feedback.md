# Melhoria Sugerida — ENH-001

## Título
Reset App State não exibe feedback visual após execução

## Descrição
Ao clicar em "Reset App State" no menu lateral, 
o carrinho é limpo corretamente, porém nenhuma 
mensagem ou indicação visual é exibida ao usuário 
confirmando que a ação foi executada com sucesso.

## Comportamento Atual
O carrinho é esvaziado silenciosamente sem nenhum 
feedback para o usuário.

## Melhoria Sugerida
Exibir uma mensagem como "Aplicativo resetado com sucesso" 
ou destacar visualmente que o carrinho foi esvaziado.

## Impacto
Sem feedback visual o usuário pode ficar confuso 
se a ação realmente funcionou ou não.
