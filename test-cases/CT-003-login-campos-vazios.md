# Caso de Teste — CT-003

## Título
Login com campos vazios

## Pré-condição
Usuário deve estar deslogado na página inicial do saucedemo.com

## Passos
1. Acessar https://www.saucedemo.com
2. Deixar o campo usuário vazio
3. Deixar o campo senha vazio
4. Clicar no botão "Login"

## Resultado Esperado
Aparecerá uma mensagem: "É necessário nome de usuário."

✅ Passou
