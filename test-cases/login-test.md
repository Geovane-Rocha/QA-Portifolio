# Teste de Login - OrangeHRM

## Cenário
Login com senha incorreta

## Passos
1. Acessar sistema
2. Inserir usuário correto
3. Inserir senha errada
4. Clicar em login

## Resultado esperado
Sistema deve exibir a mensagem: "Invalid credentials"

## Resultado obtido
Sistema exibiu corretamente a mensagem: "Invalid credentials"

## Observações
O sistema é sensível a maiúsculas e minúsculas (case-sensitive)
